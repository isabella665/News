# News Workflow

AI/comms news scanning workflow for the D8 ideas bank.

## Structure

```
workflow/
  daily-scan.md       — Morning scan checklist & log template
  weekly-review.md    — Monday check-in prep template
  google-alerts.md    — Managed alert terms & inbox setup
ideas-bank/
  YYYY-MM-DD-flags.md — Daily flagged items (one file per day)
weekly-reviews/
  YYYY-WNN.md         — Weekly review summaries
```

## Quick Start

1. Each morning: work through `workflow/daily-scan.md`
2. Flag items → create `ideas-bank/YYYY-MM-DD-flags.md`
3. Each Monday: compile flags into `weekly-reviews/YYYY-WNN.md` using `workflow/weekly-review.md`
